# Elixir - Tipos básicos

Os tipos básicos em Elixir são: integer, float, boolean, atom e string.


``` elixir
1          # integer
0x1F       # integer
1.0        # float
true       # boolean
:atom      # atom / symbol
"elixir"   # string
[1, 2, 3]  # list
{1, 2, 3}  # tuple
```
