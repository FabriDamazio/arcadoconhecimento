# Rust - Floating-Point Types (ponto flutuante)

Rust tem 2 tipos primitivos para números de ponto flutuante: `f32` (32 bits) e `f64` (64 bits). **O tipo padrão é o `f64` e ambos são signed.**

```rust
let x = 2.0 // f64
let y: f32 // f32
```
